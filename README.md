## 它是怎么工作的

核心是**20-20-20 法则**——这个东西其实早就被眼科医生推荐了很多年：

> 每用眼 20 分钟，休息 20 秒，然后眺望 20 英尺（约 6 米）外的地方。

但你自己计时是做不到的，刷着刷着就忘了。所以这个 App 帮你计时，到点了就锁屏，配合护眼引导提醒你眨眼、远眺，完成之后再自动解锁。
